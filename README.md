# PCFIX Website

Professional computer repair, troubleshooting, networking, and cybersecurity services website.

## Overview

PCFIX is a modern cybersecurity-inspired website featuring:
- Large PCFIX logo as the homepage centerpiece
- Matrix-style animated code effects
- Black and blue cyber-themed design
- Responsive layout
- GitHub Pages compatible

## Project Structure

pcfix-website/
- index.html
- pcfix_logo.png
- README.md

## GitHub Pages Deployment

1. Create a GitHub repository.
2. Upload the website files.
3. Open Settings > Pages.
4. Deploy from branch: main.
5. Select /(root).
